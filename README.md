 WallHub – Wallpaper Gallery

WallHub is a **frontend wallpaper gallery web application** that allows users to explore and preview beautiful wallpapers across multiple categories such as **For You, Animals, Nature, Mountains, and Abstract**.

The project focuses on **clean UI design, smooth interactions, and a visually rich browsing experience**, all built using pure frontend technologies.

 Features-
 
- Curated wallpaper collections by category  
- Smooth navigation with sticky navbar  
- Responsive image gallery layout  
- Full-screen wallpaper preview (modal view)  
- One-click wallpaper download option  
- Fully responsive for different screen sizes  
- Fast and lightweight (no frameworks)

 Tech Stack-
 

-HTML5 – structure and layout  
-CSS3 – modern styling, gradients, animations  
-JavaScript – dynamic rendering & interactivity
